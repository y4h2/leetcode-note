degrees * 城市被安排的数字


```python
class Solution:
    def maximumImportance(self, n: int, roads: List[List[int]]) -> int:
        degrees = [0] * n
        for road in roads:
            a, b = road
            degrees[a] += 1
            degrees[b] += 1
      
        result = 0
        degrees.sort()
        for i in range(1, n+1):
            result += degrees[i-1] * i
          
        return result
```
